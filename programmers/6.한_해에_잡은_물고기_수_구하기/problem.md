# 한 해에 잡은 물고기 수 구하기

## 1. 문제 설명

`FISH_INFO` 테이블에는 잡은 물고기의 종류, 길이, 날짜 정보가 저장되어 있다.  
이 문제에서는 **2021년 한 해 동안 잡은 물고기의 총 개수**를 계산해야 한다.

단, `TIME` 컬럼의 자료형(Date / Datetime / String)에 따라 연도 필터링 방식이 달라질 수 있다.

---

## 2. 테이블 구조

`FISH_INFO` 테이블 스키마:

| 컬럼명     | 타입     | Nullable | 설명                   |
|-----------|----------|----------|------------------------|
| `ID`      | INTEGER  | FALSE    | 물고기 ID (기본 키)    |
| `FISH_TYPE` | INTEGER | FALSE  | 물고기 종류 코드        |
| `LENGTH`  | FLOAT    | TRUE     | 물고기 길이 (NULL 가능) |
| `TIME`    | DATE / VARCHAR | FALSE | 물고기를 잡은 날짜    |

---

## 3. 요구사항

2021년 동안 잡힌 물고기 개수를 조회하라.

- 연도 기준: `2021-01-01` ~ `2021-12-31`
- 출력 컬럼명: `FISH_COUNT`
- 자료형에 따라 필터링 방식이 달라질 수 있다.

---

## 4. 자료형에 따른 처리 방식

### (1) TIME 자료형이 **DATE / DATETIME** 인 경우 (Best Practice)

날짜 함수 변환(예: `TO_CHAR`, `DATE_FORMAT`)을 WHERE 조건에서 사용하면 **인덱스가 무효화**되므로  
다음과 같이 **날짜 범위 조건**을 사용하는 것이 최적이다:
```sql
WHERE TIME >= DATE ‘2021-01-01’
AND TIME <  DATE ‘2022-01-01’
```

### (2) TIME 자료형이 **문자열(YYYYMMDD, YYYY-MM-DD)** 인 경우

문자열 비교로 연도 매칭 가능:
```sql
WHERE TIME LIKE ‘2021%’
```
이 방식은 문자열 타입일 때만 적절하다.
