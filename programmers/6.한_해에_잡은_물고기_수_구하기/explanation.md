# 한 해에 잡은 물고기 수 구하기 - 해설

## 난이도
Level 1

## 핵심 개념
- 날짜 필터링
- COUNT 집계 함수
- 자료형에 따른 처리 방식

## 접근 방식
1. TIME 컬럼에서 2021년 데이터 필터링
2. COUNT(*)로 개수 집계

## 자료형별 필터링
### DATE 타입 (권장)
- WHERE TIME >= DATE '2021-01-01' AND TIME < DATE '2022-01-01'
- 인덱스 활용 가능

### 문자열 타입
- WHERE TIME LIKE '2021%'
- 패턴 매칭 방식

## 왜 범위 조건이 좋은가?
- YEAR(TIME) = 2021 → 함수 적용으로 인덱스 무효화
- 범위 조건 → 인덱스 스캔 가능
- 대용량 데이터에서 성능 차이 큼

## 시간복잡도
O(n) - 전체 또는 인덱스 스캔
