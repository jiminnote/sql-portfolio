# 조건에 부합하는 중고거래 댓글 조회하기 - 해설

## 난이도
Level 2

## 핵심 개념
- INNER JOIN으로 테이블 연결
- 날짜 범위 필터링 (Best Practice)
- TO_CHAR로 날짜 포맷팅
- 다중 컬럼 정렬

## 접근 방식
1. 게시글과 댓글 테이블 JOIN
2. 2022년 10월 게시글만 필터링
3. 댓글 작성일을 YYYY-MM-DD 형식으로 변환
4. 댓글 작성일, 제목 순으로 정렬

## 왜 범위 조건인가?
- TO_CHAR(date, 'YYYY-MM') = '2022-10' → 인덱스 무효화
- date >= '2022-10-01' AND date < '2022-11-01' → 인덱스 활용
- 실무에서는 항상 범위 조건 권장

## 날짜 포맷팅
- TO_CHAR(R.CREATED_DATE, 'YYYY-MM-DD')
- SELECT에서만 변환 (WHERE에서는 원본 사용)

## 시간복잡도
O(n + m) - n: 게시글 수, m: 댓글 수
