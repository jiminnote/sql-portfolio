# 잡은 물고기 중 가장 큰 물고기의 길이 구하기

## 1. 문제 설명

`FISH_INFO` 테이블에는 잡은 물고기들의 ID, 종류, 길이, 날짜 정보가 저장되어 있다.  
이때, 전체 데이터 중 **가장 큰 물고기의 길이**를 구하여 `"XXcm"` 형태로 출력해야 한다.

---

## 2. 테이블 구조

`FISH_INFO` 테이블 스키마:

| 컬럼명     | 타입    | Nullable | 설명                       |
|-----------|---------|----------|----------------------------|
| `ID`      | INTEGER | FALSE    | 물고기 ID (기본 키)        |
| `FISH_TYPE` | INTEGER | FALSE  | 물고기 종류 (숫자 코드)    |
| `LENGTH`  | FLOAT   | TRUE     | 물고기 길이(cm), 일부는 NULL |
| `TIME`    | DATE    | FALSE    | 물고기를 잡은 날짜         |

**중요 포인트**

- `LENGTH` 값이 `NULL` 인 경우도 있을 수 있으나  
  `MAX()` 집계 함수는 자동으로 NULL을 무시함
- 따라서 별도의 NULL 처리가 필요 없다

---

## 3. 요구사항

아래 조건을 만족하는 SQL을 작성하라.

1. 테이블 전체에서 **가장 큰 물고기의 길이**를 조회한다.
2. 길이 값 뒤에 문자열 `'cm'` 를 붙여 `"123cm"` 형태로 출력한다.
3. 결과 컬럼명은 `MAX_LENGTH` 로 한다.

출력 예:

| MAX_LENGTH |
|------------|
| 43cm       |
