# 잡은 물고기의 평균 길이 구하기

## 1. 문제 설명

낚시 기록을 저장하는 `FISH_INFO` 테이블에서,  
현재까지 잡은 물고기들의 **평균 길이**를 구하려고 한다.

단, 물고기의 길이를 저장하는 `LENGTH` 컬럼에는 다음과 같은 규칙이 있다.

- 길이가 **10cm 이하인 물고기**는 실제 길이 대신 `LENGTH` 가 `NULL` 로 저장된다.
- 평균을 계산할 때는 이 `NULL` 값을 **10cm로 간주하여 계산**해야 한다.

이때, 조건에 맞게 **모든 물고기의 평균 길이**를 구하는 SQL을 작성하라.

---

## 2. 테이블 구조

`FISH_INFO` 테이블 스키마:

| 컬럼명     | 타입    | Nullable | 설명                     |
|-----------|---------|----------|--------------------------|
| `ID`      | INTEGER | FALSE    | 물고기 ID (기본 키)      |
| `FISH_TYPE` | INTEGER | FALSE  | 물고기 종류 (숫자 코드)  |
| `LENGTH`  | FLOAT   | TRUE     | 물고기 길이(cm), 10cm 이하는 NULL |
| `TIME`    | DATE    | FALSE    | 물고기를 잡은 날짜       |

**중요 규칙**

- `LENGTH` > 10  → 실제 길이 그대로 저장  
- 실제 길이 ≤ 10 → `LENGTH` 에 `NULL` 로 저장  
- 평균 계산 시, `NULL` 은 **10으로 변환한 뒤** 평균을 구해야 한다.

---

## 3. 요구사항

아래 조건을 만족하는 SQL을 작성하라.

1. 모든 물고기에 대해,  
   - `LENGTH` 가 `NULL` 인 경우 10으로 간주  
   - 그렇지 않으면 `LENGTH` 값을 그대로 사용
2. 위 규칙을 반영하여 전체 평균 길이를 구한다.
3. 평균 길이는 **소수점 둘째 자리까지 반올림**한다.
4. 결과 컬럼명은 **`AVERAGE_LENGTH`** 로 한다.

출력 형식 예:

| AVERAGE_LENGTH |
|----------------|
| 35.27          |
