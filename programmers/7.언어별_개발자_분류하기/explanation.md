# 언어별 개발자 분류하기 - 해설

## 난이도
Level 3

## 핵심 개념
- 비트마스크 (Bitmask)
- 비트 AND 연산으로 스킬 보유 여부 확인
- CASE WHEN으로 등급 분류
- 다중 테이블 JOIN

## 접근 방식
1. SKILL_CODE를 비트마스크로 해석
2. 특정 스킬 보유 여부를 비트 연산으로 확인
3. 조건에 따라 GRADE 부여
4. 정렬 후 출력

## 비트마스크 원리
- 각 스킬은 2의 제곱수 CODE를 가짐
- SKILL_CODE는 보유 스킬들의 CODE 합
- (SKILL_CODE & CODE) > 0 → 해당 스킬 보유

## 등급 분류 로직
- A등급: Front End + Python
- B등급: C#
- C등급: Front End만

## 왜 비트마스크인가?
- 하나의 정수로 여러 스킬 표현
- 효율적인 저장과 빠른 조회
- 실무에서 권한 관리에 자주 사용

## 시간복잡도
O(n × m) - n: 개발자 수, m: 스킬 수
