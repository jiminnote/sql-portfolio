# 601. Human Traffic of Stadium

## 문제 요약
Stadium 테이블에서 **연속된 3일 이상** 동안  
하루 방문자 수(people)가 **100명 이상**인 레코드를 모두 출력하는 문제이다.

연속 조건:
- id 컬럼이 날짜 순서를 보장한다고 가정
- people >= 100 조건을 만족하는 행만 고려

출력 조건:
- 3일 이상 연속 구간에 포함되는 모든 행 출력
- visit_date 오름차순 정렬

---

## 입력 테이블 구조

### Stadium
| Column     | Type    |
|------------|---------|
| id         | int     |
| visit_date | date    |
| people     | int     |

### 조건 정리
- `id`는 날짜 순서대로 증가한다고 가정  
- 연속된 id에서 people >= 100인 행이 3개 이상 이어지는 구간만 출력  
- 구간 전체를 출력해야 함 (즉, Top 3만 출력 아님)  

---

## 출력 예시

| id | visit_date | people |
|----|-------------|---------|
| 5  | 2017-01-05  | 145     |
| 6  | 2017-01-06  | 199     |
| 7  | 2017-01-07  | 188     |

---

## 난이도
Hard

---

## 핵심 포인트
- 연속 구간 탐지를 위한 Window Function 사용  
- “id − ROW_NUMBER()” 기법으로 연속된 id를 동일 그룹으로 묶는 패턴  
- MySQL 8 Window Function 기반의 정석적 구간 추출 문제
